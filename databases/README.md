## Projects Included

### **1. Scalable MariaDB Architecture using RDS + EC2**  
**[Open Project Folder](https://github.com/Knandwani07/aws-cloud-projects/tree/93d7865584fa4fac4f6a4a8373fb2941e0a1ad23/databases/scalable-mariadb-architecture-rds-ec2)**

A production-style database setup using **Amazon RDS (MariaDB)** connected securely to an **EC2 Linux server**.  
This project demonstrates how to deploy a managed database, configure secure networking using security groups, install SQL tools on EC2, and run SQL operations using the RDS endpoint.

This section prepares you for advanced backend architecture, production workloads, and scalable database design.
