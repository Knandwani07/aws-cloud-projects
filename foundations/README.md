# AWS Foundations

The **Foundations** section contains beginner-level AWS projects focused on core cloud concepts such as compute, identity management, security, storage, access methods, and basic cost control.  
These projects establish the skills required before moving to networking, serverless, analytics, automation, or container-based workloads.

Each project in this folder includes:
- A detailed README  
- A demo video  
- Documentation  
- Clear, practical steps  

This section reflects hands-on understanding of the AWS basics.

## Purpose of This Section

The Foundations projects aim to build:
- Confidence using the AWS Console  
- Understanding of IAM, EC2, S3, and Billing  
- Awareness of secure access practices  
- Familiarity with basic cloud administration  
- Practical hands-on experience before progressing to more advanced domains  

These projects set the groundwork for all future AWS learning.
